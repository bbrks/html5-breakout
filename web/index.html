Controls

Menu Controls

Keyboard: Up/Down to navigate, space/enter to select, Esc/Backspace to exit

Mouse: Move mouse and click to select


Game Controls

Keyboard: left/right arrows move paddle, space to launch ball

Mouse: Move mouse to move paddle, click to launch ball

Overview

This game was created for an IWP assignment, originally going to be a platformer style game, I decided to change part way through development and make a breakout/arkanoid clone

The game features multiple levels, each with unique music, with scores and lives, as well as different block types. Power-ups were planned but I ran into problems with them deleting all other objects in the game :/

No Libraries were used and all of the code and images are my own. I generated some sound effects using Bfxr and the menu/level music is free to use 8-bit remixes that I found on YouTube.
